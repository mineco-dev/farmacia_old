Six Month Calendar
  • This demo uses the bluexp theme.
  • This is a flat, in-the-page calendar.
  • This calendar displays six months, three months in each row.
  • The control month is set to the second month, which in this demo is September. The default date, all the standard controls for next month, next year, etc. are based on the control month.
  • The title changes based on the control month (titleHtml option).
